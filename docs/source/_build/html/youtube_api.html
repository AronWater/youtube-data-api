API Guide¶

The YouTube Data API has a single class YoutubeDataApi, which is authenticates your API key and provides a functions to interact with the API.

get_channel_id_from_user(username)[source]¶

Get a channel_id from a YouTube username. To get video_ids from the channel_id, you need to get the “upload playlist id”. This can be done using get_upload_playlist() for get_channel_metadata().

Read the docs: https://developers.google.com/youtube/v3/docs/channels/list

Parameters:username (str) – the username for a YouTube channel
Returns:the YouTube channel id for the given username

get_videos_from_playlist_id(playlist_id, next_page_token=None, published_after=datetime.datetime(1990, 1, 1, 0, 0), parser=<function parse_video_url>)[source]¶

Given a playlist_id, returns a list of video_ids associated with that playlist. Note that to user uploads are a playlist from channels. Typically this pattern is just the channel ID with UU subbed as the first two letters. You can access this using the function get_upload_playlist_id, or from the playlist_id_likes key returned from get_channel_metadata.

Parameters:
  • playlist_id – (str) the playlist_id IE:UUaLfMkkHhSA_LaCta0BzyhQ
  • next_page_token – (str) a token to continue from a preciously stopped query IE:CDIQAA
  • cutoff_date – (datetime) a date for the minimum publish date for videos from a playlist_id.
  • parser – (func) the function to parse the json document
Returns:

a list dictionaries with video ids associated with playlist_id.
